자바스크립트_이벤트

이벤트는 키보드를 누르거나 마우스를 클릭하는 것처럼 어떤현상이 프로그램에 영향을 미치는 것을 말합니다.

  • 마우스이벤트
  • 키보드이벤트
  • HTML 프레임이벤트
  • HTML 입력양식이벤트
  • 사용자인터페이스이벤트
  • 구조변화이벤트
  • 터치이벤트

작성법
일반적인방식 : window.onload = function(){실행문}

설명 window는 선택자의 자리, onload 는 이벤트속성이라는 이름으로 on를 제외한 load가 이벤트 이름 또는 이벤트타입이라 불리고, 이벤트 속성에 넣는 함수를 이벤트리스너 또는 이벤트핸들러 라고 한다.

추가선택자

id값을 배번 HTML 요소에 추가하는데 불편함이 있음 getElementById 값 대신에 querySelector 또는 querySelectorAll 이라는 요소로 객체로 불려낼수 있다.
이때, querySelector는 html태그 중 중복요소가 있을시 맨 처음에 있는 요소만 부르고, querySelectorAll은 모든 요소를 부른다.
